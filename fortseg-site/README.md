Fortseg - Site

Instruções rápidas:

1. Coloque a logo (arquivo que você enviou) em `public/logo-fortseg.png`.
2. Instale dependências:

   npm install

3. Rodar em desenvolvimento:

   npm run dev

4. Gerar build para produção:

   npm run build

5. Fazer upload da pasta `dist` (gerada pelo build) ou faça o deploy diretamente na Vercel (upload do projeto):

- Na Vercel: New Project → Import → Upload ZIP → Deploy

Observações:
- Substitua `public/logo-fortseg.png` pela logo de alta resolução que você me enviou.
- Se preferir, eu posso enviar o .zip pronto para upload.
